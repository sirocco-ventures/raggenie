# Ragginie Frontend


## Run Locally

Clone the project

```bash
  git clone https://github.com/sirocco-ventures/raggenie
```

Go to the project directory

```bash
  cd raggenie/ui
```

Install dependencies

```bash
  npm install
```


Create a .env file add the following env variables

```env
    VITE_BACKEND_URL=${BACKENN_URL}
```

Start the server

```bash
  npm run dev
```
